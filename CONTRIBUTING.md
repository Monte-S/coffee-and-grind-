# Contributing to Coffee and Grind

## Code Style
- Follow [Airbnb's JavaScript style guide](https://github.com/airbnb/javascript)
- Use consistent indentation (2 spaces)
- Limit lines to 80 characters

## Commit Message Conventions
- Use the present tense: "change" not "changed" or "changes"
- Limit the first line to 72 characters
- Reference issues and pull requests liberally after the first line
- Use the following format:
  
  ```
  type(scope): subject
  
  body
  
  footer
  ```

## Pull Request Process
1. Fork the repository and create your branch from `main`.
2. Ensure your code passes all tests and is thoroughly documented.
3. Open a pull request against the `main` branch.
4. Include a description of your changes and link to any relevant issues.
5. Respond to feedback promptly.

## Development Setup Instructions
1. Clone the repository:  
   ```bash
   git clone https://github.com/Monte-S/coffee-and-grind-.git
   ```
2. Change into the project directory:  
   ```bash
   cd coffee-and-grind
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Start the development server:  
   ```bash
   npm start
   ```

Thank you for contributing to Coffee and Grind! Your help is greatly appreciated!****
